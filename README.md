# Java Functional Records Exercises

This repository contains Java exercises focused on:

- records
- functional interfaces
- lambdas
- predicates
- static nested classes
- inner classes

## Exercises

### Task 1 - User Validation
Classes:
- UserForm
- UserValidator
- Main

Concepts:
- record
- Predicate
- lambda expressions

### Task 2 - Service Price Calculator
Classes:
- ServiceOrder
- PriceStrategy
- PriceCalculator
- Main

Concepts:
- @FunctionalInterface
- lambdas
- strategy behavior

### Task 3 - Store Order
Classes:
- Order
- Order.OrderItem
- OrderSummary
- Main

Concepts:
- static nested class
- record

### Task 4 - Library Account
Classes:
- LibraryAccount
- FineCalculator
- MessagePrinter
- Main

Concepts:
- inner class
- lambda
- anonymous behavior

- Java 17+
